# Lottie Animations Folder

LottieFiles.com no longer allows direct CDN hotlinking for free animations (returns 403 Forbidden).

To enable the animations in the **Benefits** section:
1. Go to [LottieFiles.com](https://lottiefiles.com/) (or any Lottie provider).
2. Download your preferred **Lottie JSON** animations.
3. Rename the files and place them in this folder:
   - `shield.json`
   - `cut.json`
   - `eye.json`
   - `castle.json`

Once added, Astro will automatically serve them to the `<lottie-player>` components.
